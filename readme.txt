Introduction:

The first thing to use this equalizer is push button "input original sound".
And then, the waveform will be displayed.

A scroll bar (which above button "play original sound")to select a piece of sound. 
Considering that some sound files may be very large. 
So,you can use this scroll bar to select the sound form beginning to relative sound position. 

The button "filter" is to filter the sound you select.
